An implementation of BaseBack

## Dependencies
- php: ^7.2.5|^8.0
- fideloper/proxy: ^4.4
- fruitcake/laravel-cors: ^2.0
- guzzlehttp/guzzle: ^7.0.1
- laravel/framework: ^8.0
- laravel/legacy-factories: ^1.3
- laravel/tinker: ^2.5
- tymon/jwt-auth: dev-develop

## Install
>composer install

>php artisan migrate

>php artisan db:seed

>php artisan jwt:secret

>php artisan key:generate
